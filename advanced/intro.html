# 介绍

Waline - 一款从 Valine在新窗口打开 衍生的带后端评论系统。可以将 Waline 等价成 With backend Valine.

# 特性

  • 快速
  • 真·安全
  • Markdown 语法支持
  • 轻量易用
  • 免费部署
  • 多种部署部署方式和存储服务支持，每列选择一项多达 84 种部署方式任君选择
